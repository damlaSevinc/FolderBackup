# FolderBackup

### Description

Folder Backup is a simple tool to backup your files and folders. It is written in Python. It works 
recursively.

### Example Usage

```
python folderbackup.py -s test/test_src/ -d test/test_dst/ -i 500 -l mylog.log
```
